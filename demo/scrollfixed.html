Fixed Columns

This page shows how to configure Handsontable fixed columns:

  • One fixed column
  • Two fixed columns

One fixed column

Specify one fixed column with fixedCols: 1 option.

Note: You'll need horizontal scrollbars, so just set a container width and overflow: scroll in CSS.

Two fixed columns

Specify two fixed columns with fixedCols: 2 option.

Note: You'll need horizontal scrollbars, so just set a container width and overflow: scroll in CSS.
